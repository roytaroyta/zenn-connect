---
title: "04 ローカル実行ガイド"
---

ローカルでの実行手順、デバッグ方法、よく使うコマンドをまとめます。

基本コマンド:

```bash
npm ci
npm run build
npm start
```

デバッグ:

- `ts-node` を使った直接実行
- 環境変数の切り替え（.env ファイル + dotenv）
- ログレベルの設定方法

dry-run の実行例:

- Canary ワークフローで使われる `dry_run=true` オプションに相当するローカルフラグの使い方を記載

トラブルシュート:

- よくあるエラーと対処法（依存関係の不一致、環境変数の不足）
